```markmap
# 广东专插本报名资格2024年

## 是高考前是广东省户籍

- 是全日制大专
    - 大一、大二
        - 不能报名
    -  应届毕业生
        - 可以报名
    - 往届毕业生
        - 广东省户籍是连续满3年
            - 可以报名
        - 广东省户籍不连续满3年
            - 广东参加高考
                - 参加高考时是广东户籍
                    - 可以报名
                - 参加高考时是其他省户籍
                    - 不能报名
- 非全日制大专
    - 广东省户籍是连续满3年
        - 毕业证书日期在2024年2月底前
            - 有基本医疗保险+基本养老
                - 缴费累计满1年
                    - 可以报名
                - 缴费累计无1年
                    - 不能报名
            - 无基本医疗保险+基本养老
                - 不能报名
        - 毕业证书日期在2024年2月底后
            - 不能报名  
    - 广东省户籍不连续满3年
    - 不能报名

## 非广东省户籍

- 是全日制大专
    - 应届毕业生
        - 就读于广东省的大专
            - 可以报名
        - 就读于其他省的大专
            - 高考是在广东省参加的
                - 可以报名
            - 高考是在其他省参加的
                - 不能报名
- 非全日制大专
    - 往届毕业生
        - 不能报名

```
